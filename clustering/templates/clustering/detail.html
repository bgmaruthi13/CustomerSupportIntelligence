Help — Understanding this cluster
  • Confidence Score — a blend of cluster size relative to the run's largest cluster, membership density (how tightly HDBSCAN thinks these tickets belong together), and how recent the tickets are. Higher = more reliable grouping.
  • Trend — compares ticket count in the earliest third of this cluster's date range against the latest third. Hover the trend badge to see the exact counts and windows compared.
  • Top Application / Top Offering / Top Country — the single most common value among this cluster's member tickets (Offering counts each value on a multi-valued ticket separately).
  • Verdict — restates whether this cluster currently clears the Problem Candidate bar. See the Help section on the Problem Clusters list page for the full logic and a worked example.

Member Tickets are sorted by similarity — the tickets HDBSCAN is most confident belong in this cluster appear first.
