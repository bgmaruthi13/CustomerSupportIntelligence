Help — How does Ask Correlate work?

Your query is embedded with the same local sentence-embedding model used by the Generative AI clustering pipeline, then ranked against every ticket's stored embedding by cosine similarity — the same technique behind semantic search. This is retrieval, not generation: results are ranked existing tickets, not an AI-written answer, and nothing is sent to an external API.

Tickets that have an embedding but aren't currently in a Generative AI cluster show as "Not yet clustered" rather than being hidden from results.
